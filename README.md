# DoomLike

Godot 4.6 制作的 DOOM 风格第一人称射击游戏。

## 特性

- **快节奏 FPS 战斗** — hitscan/近战/弹道武器，四槽位切换
- **Counter 系统** — 敌人攻击中受击触发眩晕暴增 + 青蓝闪白，打断攻击
- **铁鞭连招** — 滚轮挥鞭 → 拉取抓取 → 举盾格挡 → 冲刺处决 / F 处决
- **硬锁定 ADS** — 右键自动追踪最近敌人，50m 范围、20° FOV
- **16 状态敌人 AI** — 五档距离判定，护甲系统，跳跃/防御/倒地/被抓取
- **12 种敌人** — 地面/飞行/远程/兽人战士等，分阶随难度入池
- **20 级强度曲线** — 0~20 分钟逐步升级，刷怪间隔从 4s 缩至 0.6s
- **局内成长** — 击杀获经验，升级三选一卡片，12 种升级叠加
- **双关卡** — 沙漠竞技场 / 熔岩地狱（昼夜循环）
- **多平台** — PC / Android / Web 三平台导出

## 运行

1. 安装 [Godot 4.6](https://godotengine.org/)
2. `git clone` 本项目
3. 用 Godot 打开 `project.godot`
4. 按 F5 运行

## 操作

| 按键 | 功能 |
|------|------|
| WASD | 移动 |
| 鼠标 | 视角 |
| 左键 | 主武器攻击 |
| 右键 | 硬锁定 ADS |
| 滚轮 | 铁鞭攻击/冲刺处决 |
| Shift | 冲刺 |
| 1-4 | 切换武器 |
| R | 换弹 |
| F | 处决/副武器 |
| Space | 跳跃 |
| Esc | 暂停 |

## 技术栈

- **引擎**：Godot 4.6 (Forward Plus, D3D12)
- **物理**：Jolt Physics
- **语言**：GDScript
- **分辨率**：1280×720

## 目录结构

```
scripts/         游戏脚本
  core/          核心系统（状态机/信号总线/存档）
  player/        玩家控制
  weapon/        武器系统（4 枪 + 铁鞭）
  enemy/         敌人系统（基类 + 12 种敌人 + 刷怪）
  ui/            UI（HUD/菜单/升级面板/小地图）
  progression/   局内成长（经验/等级/升级卡片）
  level/         关卡（沙漠/熔岩/随机化/昼夜循环）
  platform/      多平台适配（触摸/UI缩放/Web导出）
scenes/          场景文件
assets/          资源（武器数据/敌人数据/升级/材质/标题图）
```
