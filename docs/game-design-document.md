# DoomLike 游戏设计文档

> Godot 4.6 · Jolt Physics · Forward+ D3D12 · 1280×720

---

## 一、游戏概览

**DoomLike** 是一款快节奏第一人称射击游戏，核心玩法为：**枪械破甲 → 铁鞭眩晕 → 抓取处决**。玩家在圆形竞技场中对抗波次刷新的恶魔敌人，通过击杀获取分数和掉落补给，尽可能延长生存时间。

| 属性 | 值 |
|------|-----|
| 引擎 | Godot 4.6 (Forward Plus, D3D12) |
| 物理 | Jolt Physics 3D |
| 分辨率 | 1280×720 |
| 主场景 | `res://scenes/main.tscn` |

---

## 二、游戏流程

```
启动 → 主菜单 → 选关 → 战斗(PLAYING) ←→ 暂停(PAUSED)
                         ↓ 玩家死亡
                       结算(GAME_OVER)
                         ↓
                   重新开始 / 选关 / 主菜单
```

**关卡**：

| 关卡 | 名称 | 主题 | 雾颜色 | 场景特色 |
|------|------|------|--------|----------|
| desert | 第一关：荒漠 | 沙漠荒原 | 沙黄 | 24棵枯树掩体 |
| lava | 第二关：熔岩地狱 | 熔岩地狱 | 暗红 | 18根岩柱 + 2条熔岩河(18DPS) |

## 五、敌人系统

### 5.1 基类系统

**16状态机**：SPAWNING → IDLE → CHASE/WALKING/RUNNING → ATTACK_PREPARE → ATTACK_ACTIVE → ATTACK_RECOVER → (循环)  
受击分支：PAIN / STUNNED / GRABBED / KNOCKED_DOWN / EXECUTED / DEATH  
特殊：DEFENDING（兽人举盾）

**距离开关**（全局统一）：

| 档位 | XZ距离 | 含义 |
|------|--------|------|
| MELEE | <1m | 贴身 |
| CLOSE | 1~3m | 近距离 |
| MEDIUM | 3~8m | 中距离 |
| FAR | 8~25m | 远距离 |
| SUPER_FAR | >25m | 超远距离 |

**核心机制**：
- **眩晕**：累积至 `max_stun` → 进入 STUNNED 状态，可被抓取。自然恢复 `stun_recovery_rate`/s
- **击退**：`有效力 = 力 / max(重量, 0.1) × (1-击退抗性)`
- **护甲**（敌人）：1:1吸收伤害，每实例独立
- **Counter**：攻击阶段受击 → 2倍眩晕 + 攻击被打断
- **增伤标记**：`apply_damage_mark(持续时间, 倍率)`，下次受击消耗

### 5.3 兽人特殊AI（唯一有护甲的敌人）

```
SUPER_FAR >25m → RUNNING 冲刺
FAR     8~25m → RUNNING 冲刺
MEDIUM   3~8m → WALKING/多敌→DEFENDING+侧翼包抄
CLOSE    1~3m → DEFENDING举盾 + 40%概率斧击（主战斗距离）
MELEE    <1m  → 后退+DEFENDING（不进入贴身）
```

攻击三段式：举斧发光(1.0s) → 挥砍判定(0.1s) → 收斧恢复(0.5s)，CD 2.0s。判定框 1.2m³ 在身前1.2m处。

## 九、已知问题

1. **CSGBox3D 调试条偏暗**：敌人头顶血条/眩晕条在无直接光照下材质偏暗
2. **Godot MCP validate_script 误报**：大量脚本报 line 0 语法错，analyze_script 和实际运行正常
3. **`_can_see_player()` 死代码**：完整视线检测函数未使用，敌人可穿墙感知
4. **`_sight_ray` 死节点**：RayCast3D 创建后从未使用
5. **投射物挂在 `get_tree().root`**：关卡切换时投射物不会被清理（等 lifetime 到期）
6. **drop_manager 在 Main 下**：非 Level 子节点，关卡卸载时需手动释放
7. **StandardEnemy 缺少 `max_stun` 配置**：使用默认值而非显式设定
8. **只有兽人有护甲**：破甲→处决核心循环仅对兽人生效
