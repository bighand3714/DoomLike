# 手动测试指南

> 按顺序执行，每项通过后打勾。预计 25-30 分钟完成全部。

---

## 准备工作

1. 在 Godot 编辑器中打开项目
2. 打开 `res://scenes/levels/test_arena.tscn`（测试关卡）
3. 确保场景中有 PlayerStart 节点（玩家出生点）

## 快速检查：只测 Bug Fix（~10分钟）

如果时间有限，只跑这几项：

1. **拳头连击** → 切拳头最快速度连点，确认不抖动
2. **StandardEnemy 跳跃** → 站着吃跳跃攻击，确认只掉 20HP
3. **兽人双重伤害** → 贴脸吃斧击，确认单次伤害
4. **护甲独立** → 放 2 个兽人，鞭打A破甲，确认B护甲满
5. **HUD 通知** → 触发波次/盾牌通知，确认会自动消失
6. **方向指示器** → 不同朝向受击，确认箭头方向正确
7. **处决不崩溃** → 抓敌→R处决，确认无报错
8. **防御移动** → 中距离看兽人是否横移（不再原地不动）
